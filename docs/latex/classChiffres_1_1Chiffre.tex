\doxysubsection{Detailed Description}
Core abstraction of the Chiffres monetary system. 

A Chiffres wallet is a living value container with two fundamental modes\+:
\begin{DoxyItemize}
\item In circulation, transactions are exact and deterministic.
\item At rest, the value undergoes probabilistic mutations governed by\+:
\begin{DoxyItemize}
\item Ethical influence (\doxylink{structChiffres_1_1EthicsScore}{Ethics\+Score})
\item Natural cycles (\doxylink{structChiffres_1_1CyclePhase}{Cycle\+Phase})
\item Collective field feedback (\doxylink{classChiffres_1_1Ether}{Ether})
\item Its own memory of past behaviors (\doxylink{structChiffres_1_1Memory}{Memory})
\end{DoxyItemize}
\end{DoxyItemize}

The class encapsulates both deterministic accounting and stochastic dynamics.

\item \doxylink{structChiffres_1_1Memory}{Memory} (feedback from past events)
\item \doxylink{classChiffres_1_1Ether}{Ether} (collective field density, self-\/regulation)
