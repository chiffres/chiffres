\doxysubsection{Detailed Description}
Collective monetary field in the \doxylink{classChiffre_1_1Chiffres}{Chiffres} system. 

\doxylink{classChiffre_1_1Ether}{Ether} represents the "{}atmosphere"{} of value shared by all \doxylink{classChiffre_1_1Chiffres}{Chiffres}. It reacts to individual gains and losses, and in turn influences the probability space of all future mutations.

Core principles\+:
\begin{DoxyItemize}
\item When a wallet gains in rest, \doxylink{classChiffre_1_1Ether}{Ether} is "{}consumed"{} → density decreases, value of all \doxylink{classChiffre_1_1Chiffres}{Chiffres} increases slightly (rarer field).
\item When a wallet loses in rest, \doxylink{classChiffre_1_1Ether}{Ether} "{}dissolves"{} → density increases; The value of all \doxylink{classChiffre_1_1Chiffres}{Chiffres} decreases slightly (thicker field).
\end{DoxyItemize}

\doxylink{classChiffre_1_1Ether}{Ether} acts as a global coupling mechanism\+: no mutation is purely individual. The system becomes organically self-\/regulating.
