\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Vision}{Vision}}\label{md_README_autotoc_md1}
{\bfseries{Chiffres}} is not just a currency – it is a {\bfseries{living system}}. Unlike traditional money (stable when hoarded, inert when circulating), Chiffres reverses the paradigm\+:


\begin{DoxyItemize}
\item {\bfseries{Circulation = Certainty}} When you spend or receive, the value is exact. 100 Chiffres spent = 100 Chiffres received. But {\itshape how} you spend matters\+:
\begin{DoxyItemize}
\item {\bfseries{Local good}} → stabilizes your Chiffres.
\item {\bfseries{Risky project}} → makes them more volatile in the future. Thus, the currency {\bfseries{remembers how it has circulated}}.
\end{DoxyItemize}
\item {\bfseries{Rest = Gamble}} When hoarded, Chiffres enter a quantum-\/like state. Their value mutates unpredictably according to\+:
\begin{DoxyItemize}
\item {\bfseries{Ethics / GEMs}} (virtuous actions stabilize, destructive ones destabilize).
\item {\bfseries{Cycles of Time}} (solstices, seasons, years).
\item {\bfseries{Weights / Elements}} (small fortunes = wind/water, large fortunes = earth/fire).
\item {\bfseries{Collective Ether}} (if many stagnate, all are dragged down; if all circulate, everyone rises).
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Core Principles}{Core Principles}}\label{md_README_autotoc_md2}

\begin{DoxyEnumerate}
\item {\bfseries{Circulation as Memory}}
\begin{DoxyItemize}
\item Spending leaves an imprint. The past of your Chiffres influences their volatility.
\end{DoxyItemize}
\item {\bfseries{GEMs as Catalysts}}
\begin{DoxyItemize}
\item Linked to concrete actions (planting a tree, donating, collaborating).
\item Reduce losses or amplify gains in the cosmic gamble.
\end{DoxyItemize}
\item {\bfseries{Temporal Cycles}}
\begin{DoxyItemize}
\item Mutations follow natural rhythms.
\item At the winter solstice, great fortunes face monumental risks.
\item At equinoxes, the system seeks balance.
\end{DoxyItemize}
\item {\bfseries{The Collective Ether}}
\begin{DoxyItemize}
\item Gains “consume” the Ether (making Chiffres rarer, everyone benefits).
\item Losses “dissolve” into the Ether (making it thicker, everyone loses a bit).
\item When too many stagnate, value decreases → inciting action.
\end{DoxyItemize}
\item {\bfseries{Symbolic Weights}}
\begin{DoxyItemize}
\item {\bfseries{Light weights (small sums)\+:}} wind, water – fast, playful, changeable.
\item {\bfseries{Heavy weights (large sums)\+:}} earth, fire – slow, transformative, existential.
\end{DoxyItemize}
\item {\bfseries{Council of Chiffres (Governance)}}
\begin{DoxyItemize}
\item Open assembly adjusting parameters (mutation frequency, Ether weights).
\item Decisions themselves are gambles – they may backfire if the Ether shifts.
\end{DoxyItemize}
