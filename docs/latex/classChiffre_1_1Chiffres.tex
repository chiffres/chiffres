\doxysubsection{Detailed Description}
Core abstraction of the \doxylink{classChiffre_1_1Chiffres}{Chiffres} monetary system. 

A \doxylink{classChiffre_1_1Chiffres}{Chiffres} wallet is a living value container with two fundamental modes\+:
\begin{DoxyItemize}
\item In circulation, transactions are exact and deterministic.
\item At rest, the value undergoes probabilistic mutations governed by\+:
\begin{DoxyItemize}
\item Ethical influence (\doxylink{structChiffre_1_1EthicsScore}{Ethics\+Score})
\item Natural cycles (\doxylink{structChiffre_1_1CyclePhase}{Cycle\+Phase})
\item Collective field feedback (\doxylink{classChiffre_1_1Ether}{Ether})
\item Its own memory of past behaviors (\doxylink{structChiffre_1_1Memory}{Memory})
\end{DoxyItemize}
\end{DoxyItemize}

The class encapsulates both deterministic accounting and stochastic dynamics.

\item \doxylink{structChiffre_1_1Memory}{Memory} (feedback from past events)
\item \doxylink{classChiffre_1_1Ether}{Ether} (collective field density, self-\/regulation)
